复杂前端交互

可以做搜索、筛选、折叠、图表和多页导航；但不适合私密 API Key 或数据库写入。

后续推荐用法

  1. 1普通一次性审阅：我直接生成单文件 HTML，你在 ChatGPT 里打开。
  2. 2需要分享或长期在线：我更新到 HisenWeb/html-preview 的 docs/ 目录。
  3. 3多个报告：用 docs/reports/*.html 存报告，再维护一个目录页。
  4. 4复杂交互：仍然保持静态前端，数据内联或放 JSON 文件，避免后端依赖。
